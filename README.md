# Cybersecurity-Work-Force-Certification-Training
This is a complete look at my training with Ivy Tech Community College
## Program Overview

 The CWCT System Administration track is designed to build foundational and advanced skills in computer systems,networksm and cybersecurity, with a focus on managing critical infrastructure, analyzing vulnerabilities, and protecting systems.

## Core Certification Requirements
   All CWCT tracks required 3 core certifications:

   * CWCT100 – CompTIA A+ (45 hours) — Covers hardware, software, networking basics, OS installation/config, troubleshooting, and cloud fundamentals Github.

   * CWCT105 – CompTIA Security+ (45 hours) — Focuses on cybersecurity principles, cryptography, threats, vulnerabilities, access control, and security across networks, applications, data, and hosts m-soro.github.io.

   * CWCT115 – Cisco CyberOps (45 hours) — Prepares for associate-level cybersecurity analyst roles, covering monitoring, detection, analysis, and response to cyber threats 
## Additional System Administration Courses
  
   * CWCT116 – Linux System Administration (45 hours) — Linux configuration, management, and security

## Track I chose...
I chose the digital forensics route. This track gave me 4 courses that cover the fundamentals of computer systems, networks, digital forensics investigation, and advanced topics in Windows and cell phone forensics. I completed Cisco Networking Academy courses.
